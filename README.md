# Cnake

This is a game of snake made in C using Raylib

## Controls

| input | output |
| ----- | ------ |
| W | moves snake up |
| S  | moves snake down |
| D  | moves snake right |
| A  | moves snake left |

## Build

Run the following to build
```
cmake -B build
cmake --build build
```
Then run cnake in the build folder to play
